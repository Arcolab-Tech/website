# Arcolab Tech website

Site institucional estatico para `https://tech.arcolab.com.br`.

## Estrutura

- `index.html`: pagina principal
- `privacy.html`: politica de privacidade simples
- `support.html`: canal de suporte
- `styles.css`: identidade visual e responsividade
- `script.js`: ano automatico no rodape
- `assets/arcolab-institutional-bg.webp`: background institucional otimizado
- `assets/arcolab-institutional-bg.png`: fallback e imagem social

## Publicacao

O site pode ser publicado em qualquer hospedagem estatica. Para passar pelo
cadastro da Apple, confirme antes:

- `https://tech.arcolab.com.br` abre publicamente com HTTPS.
- `contato@arcolab.com.br` existe e recebe e-mails.
- O DNS do subdominio aponta para a hospedagem escolhida.
- As paginas `privacy.html` e `support.html` abrem sem login.

O background institucional veio do material usado nas reunioes do Teams. A
paleta principal aplicada no CSS parte de `#07326a`, `#104680`, `#156baa`,
`#1772b0` e branco.
