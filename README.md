# ARCOLAB Tech website

Site institucional estático para `https://tech.arcolab.com.br`.

## Estrutura

- `index.html`: página principal
- `privacy.html`: política de privacidade simples
- `support.html`: canal de suporte
- `styles.css`: identidade visual e responsividade
- `script.js`: ano automático no rodapé
- `assets/arcolab-institutional-bg.webp`: background institucional otimizado
- `assets/arcolab-institutional-bg.png`: fallback e imagem social

## Publicação

O site pode ser publicado em qualquer hospedagem estática. Para passar pelo
cadastro da Apple, confirme antes:

- `https://tech.arcolab.com.br` abre publicamente com HTTPS.
- `contato@arcolab.com.br` existe e recebe e-mails.
- O DNS do subdomínio aponta para a hospedagem escolhida.
- As páginas `privacy.html` e `support.html` abrem sem login.

O background institucional veio do material usado nas reuniões do Teams. A
paleta principal aplicada no CSS parte de `#07326a`, `#104680`, `#156baa`,
`#1772b0` e branco.
